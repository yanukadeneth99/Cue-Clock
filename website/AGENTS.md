## Core Value Proposition
Cue Clock is a specialized timing tool for broadcast production. It addresses the need for a minimal, distraction-free, and reliable clock app that handles multiple timezones and countdowns without the complexity and cost of enterprise solutions.

## Key Technical Specifications
- **Multi-Zone Support**: Configurable for up to 18 broadcast timezones.
- **Countdown Engine**: Simultaneous readout of infinite timers.
- **Broadcast Specific**: Includes "Deduction Offsets" for pre-show buffers.
- **Offline First**: Uses local storage for persistent state without cloud dependencies.
- **Zero Cost**: No ads, no subscriptions, fully open-source.

## Architecture
- **Frontend**: Next.js (Website), React (Web App).
- **Mobile**: React Native / Expo.
- **Animations**: GSAP (GreenSock Animation Platform) for high-performance UI motion.
- **Styling**: Tailwind CSS for modern, responsive layouts.

## For Developers
Developers can contribute to the project on GitHub. The codebase is structured into a `website` (Next.js) and `app` (React Native) directory.
